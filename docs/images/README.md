# GrinSh Logo Assets

## Logo File

The GrinSh logo should be placed in this directory as `logo.svg` or `logo.png`.

**Source**: https://chatgpt.com/s/m_69212b877200819196e14133ac336106

## Usage

Once added, the logo will be automatically displayed in the site header by `index.html`.

## Manual Steps Required

1. Visit the ChatGPT share link above
2. Download the logo image
3. Save it as `docs/images/logo.svg` (preferred) or `docs/images/logo.png`
4. Update `docs/index.html` if using PNG instead of SVG

## Brand Guidelines

Follow the GrinSh brand guidelines when creating or modifying logo assets:
- Use Mischief Lime (#8DF55E) as primary brand color
- Maintain dark theme compatibility
- Incorporate Seuss-inspired playful energy
