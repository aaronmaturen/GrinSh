GrinSh

A conversational shell for macOS

About

GrinSh is a conversational shell for macOS that understands natural language. No flags, no man pages, no syntax to remember. Just say what you want, and grinsh does it.

Examples

> what's using port 8080
> compress the projects folder
> find that pdf I downloaded last week
> quit slack

Features

  • Natural Language: Just say what you want in plain English. No syntax to remember.
  • Native macOS: Built with Swift, using native macOS APIs for files, apps, and system operations.
  • Auto-installing Tools: Discovers and installs CLI tools via Homebrew on demand.
  • Conversation Memory: Remembers context across your session for smarter interactions.
  • Touch ID Support: Privileged operations use native macOS authentication.
  • Escape Hatch: Run raw commands with !command when you need it.

Installation

Quick Install (Recommended)

curl -sSL https://raw.githubusercontent.com/aaronmaturen/GrinSh/main/install.sh | bash

This will download the latest release, verify checksums, and install to /usr/local/bin

Prerequisites

  • macOS 13.0 or later
  • Claude API key from console.anthropic.com
  • Homebrew (recommended, for CLI tool installation)

Next Steps

  1. Add your Claude API key to ~/.grinshrc
  2. Run grinsh to start
  3. (Optional) Set as default shell with chsh -s /usr/local/bin/grinsh

How It Works

  1. You speak: Type your request in natural language
  2. Claude interprets: AI understands your intent and finds the right tool
  3. GrinSh executes: Native APIs or CLI tools get the job done

Built-in tools use native Swift APIs. Unknown tools are discovered via Homebrew, learned from documentation, and cached for future use.
