GrinSh

A conversational shell for macOS

Stop memorizing commands.
Start talking.

No flags, no man pages, no syntax to remember. Just say what you want, and grinsh does it.

grinsh
> what's using port 8080
> compress the projects folder
> find that pdf I downloaded last week
> quit slack

Installation

Quick Install (Recommended)

curl -sSL https://raw.githubusercontent.com/aaronmaturen/GrinSh/main/install.sh | bash

This will download the latest release, verify checksums, and install to /usr/local/bin

Prerequisites

  • macOS 13.0 or later
  • Claude API key from console.anthropic.com
  • Homebrew (recommended, for CLI tool installation)

Next Steps

  1. Add your Claude API key to ~/.grinshrc
  2. Run grinsh to start
  3. (Optional) Set as default shell with chsh -s /usr/local/bin/grinsh
